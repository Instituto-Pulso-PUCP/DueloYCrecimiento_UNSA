## Descripción del Proyecto

El estudio "Pérdida, Duelo y Crecimiento" es parte de un proyecto de investigación que tiene como objetivo analizar las experiencias de pérdida y explorar los recursos y estrategias que se vienen desplegando por parte de las y los estudiantes universitarios, en el contexto de pérdidas y duelos a nivel nacional por la pandemia de la COVID-19.

## Estructura del Repositorio

- **`code/`**: Contiene los archivos SPSS (.sps) utilizados para el análisis de datos
- **`docs/`**: Documentación complementaria del proyecto

## Acceso a Datos

Los datos no están incluidos en este repositorio debido a consideraciones de confidencialidad y protección de información personal de los participantes. Para acceder a los datos o solicitar información adicional sobre el estudio, contactar a **Pulso PUCP:**  `PulsoPucp@pucp.edu.pe`
